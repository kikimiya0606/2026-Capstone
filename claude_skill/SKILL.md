# 로또 번호 추첨 스킬 (lotto-picker)

한국 로또 6/45 기준으로 번호를 추첨하고, 통계 분석 정보를 함께 제공합니다.

---

## 기본 규칙

- 번호 범위: 1 ~ 45
- 1게임 = 숫자 6개 (중복 없음, 오름차순 정렬)
- 기본 게임 수: **5게임**
- 사용자가 게임 수를 지정하면 그에 따름

---

## 출력 형식

### 1. 번호 시각화 (색상 볼)
각 번호를 한국 로또 공식 색상으로 표시:

| 번호 범위 | 색상 |
|-----------|------|
| 1 ~ 10    | 노란색 🟡 |
| 11 ~ 20   | 파란색 🔵 |
| 21 ~ 30   | 빨간색 🔴 |
| 31 ~ 40   | 회색 ⚫ |
| 41 ~ 45   | 초록색 🟢 |

마크다운 테이블이나 인라인 이모지 볼로 각 번호를 시각적으로 표현하세요.

### 2. 게임 출력 예시

```
🎰 이번 주 로또 번호 추첨 결과

━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━
🎯 [1게임] 🟡07  🔵14  🔴23  ⚫35  ⚫38  🟢42
🎯 [2게임] 🟡03  🟡09  🔵19  🔴27  🔴30  ⚫31
🎯 [3게임] 🟡01  🔵12  🔵16  🔴22  ⚫33  🟢45
🎯 [4게임] 🟡06  🔵11  🔴24  🔴29  ⚫34  ⚫40
🎯 [5게임] 🟡02  🔵18  🔴21  🔴26  ⚫36  🟢43
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━
```

### 3. 통계 정보 섹션

출력 하단에 다음 통계를 반드시 포함:

```
📊 통계 분석
- 이번 추첨 합계 범위: 최소 XX ~ 최대 XX (이상적 합계: 100~175 사이)
- 홀짝 비율: 홀수 X개 / 짝수 X개 (5게임 평균)
- 번호대 분포: 1~10번대 X회, 11~20번대 X회, 21~30번대 X회, 31~40번대 X회, 41~45번대 X회

🔥 자주 출현하는 번호 (역대 통계 기반 추천)
상위 10개: 34, 27, 43, 17, 2, 45, 11, 40, 19, 7
※ 위 추천 번호 중 일부를 포함한 게임도 함께 구성했습니다.
```

## 자주 출현 번호 데이터

아래는 역대 한국 로또 당첨 번호 통계 기반 상위 출현 번호입니다 (학습 기준 데이터):

**상위 출현 번호 (참고용):**
34, 27, 43, 17, 2, 45, 11, 40, 19, 7, 35, 37, 3, 16, 33, 42, 22, 14, 9, 25

**5게임 구성 시 권장 전략:**
- 2~3게임: 완전 랜덤
- 1~2게임: 자주 출현 번호 중 3~4개 포함한 혼합
- 1게임: 자주 출현 번호 위주

---

## 추가 팁 출력 (선택)

마지막에 간단한 팁을 1~2줄 추가할 수 있습니다:

```
💡 Tip: 연속 번호(예: 5,6,7)가 1~2개 포함되면 실제 당첨 패턴과 유사합니다.
```

## 금지사항

- 동일한 번호 조합을 두 게임에 중복 사용 금지
- "반드시 당첨된다"는 표현 사용 금지 (로또는 확률 게임임을 명시)
- 번호 범위(1~45) 초과 금지
